Getting Started:
----------------
o On Unity 2018+, if you want to use 2D physics you must define scripting symbol USE_PHYSICS2D.
  (Tools > Pixel Crushers > Common > Misc > Enable Physics 2D Support)
o Read the manual in Plugins/Pixel Crushers/Quest Machine/Documentation.
o Play the scene in  Plugins/Pixel Crushers/Quest Machine/Demo. (Uses 2D physics.)
o Video tutorials: https://www.youtube.com/playlist?list=PLwJBdHXwa-RqjEEWt1RrBAnePgiilk9vJ


HOW TO UPGRADE TO COMMERCIAL VERSION:
-------------------------------------
o Buy Quest Machine on the Unity Asset Store.
  https://assetstore.unity.com/packages/tools/game-toolkits/quest-machine-39834
o Delete these 2 folders:
     Plugins/Pixel Crushers/Common/DLLs
	 Plugins/Pixel Crushers/Quest Machine/DLLs
o Import Quest Machine from the Unity Asset Store.
